The first，博客搭建记录

Written by 默从心 with ♥ on April 18, 2019 in 生活 其他 , 690 words

对人生第一次搭建博客的一个记录。

为什么搭建博客

随着对编程的学习越来越深入，感觉需要一个这么样的博客来记录自己理解和学习过程。于是就想着，与其用其他平台的，不如自己搭建一个，于是这个博客就诞生了。

事实上，这篇文章是在搭建完博客之后一段时间才完成的懒癌晚期没救了，接下来就梳理一下搭博客的心路历程。

首先是需求，需求是如何产生的呢？大抵是需要有一个能够记录自己的学习笔记、idea、心情以及其他琐琐碎碎的东西的地方，于是就产生了搭建个博客的念头。然后就是搭建方案了，有想过自己写个博客生成器，但仔细想想，一来比较麻烦，二来这么搞好像重心不太对，于是放弃。那么接下来就选哪个的问题了。

wordpress

过于重量，虽然很方便，但我需要的就是一个静态页用不着这个体量的框架，而且对 PHP 也不熟，貌似还收费？所以 pass。

ghost

轻量、简洁、方便、默认主题也很好看，很对我的胃口，但是（万事就怕但是），ghost 是做动态博客的不能直接部署到 github page 上，然后查了一下，转静态坑很多，无奈放弃。

hugo

感觉挺好用，但不知道为什么当初没有选它。 现在正在用。

jekyll

github 官方支持，但是名字太难念，加上对 ruby 不熟，也 pass。

hexo

我最终的选择，没有什么特别的原因，就是把其他的排除了而已。

选完框架，接下来就是选主题了，据说用 hexo 的有一半都是用 next，我也试了一下，next 提供了三款主题，都很简洁、大气、有设计感，但是（还是但是），不太符合我的审美，所以最终选择了 apollo。

为什么换成 Hugo 呢

也没什么特别的原因，因为换了台新电脑，搭建环境的时候想着不如直接换 Hugo 吧，就换成这个了。准备在这个博客记录一些学习笔记、技术文章、以及自己画的画，希望能够坚持吧。
